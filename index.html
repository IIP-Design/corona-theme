General Markup Structure

See the Visual Markup Guide to see how Corona's markup is structured.

The markup structure was heavily influenced by the Genesis Framework, and the Visual Markup Guide is forked from Nathan Rice's invaluable Genesis Visual Markup Guide. It has been updated to reflect the changes we've made to Genesis' markup structure.

Development Setup

Build Tools

Corona uses NPM Scripts to run its build process. Checkout the package.json file to see what's included and version numbers. Here are the highlights:

  • SASS
  • PostCSS + autoprefixer. We support browsers that are >5%. See Browserlists.
  • ESLint + AirBnb's JavaScript style guide.
  • UglifyJS
  • Imagemin
  • BrowserSync

In development, you should be able to watch for changes and recompile your assets by running:

$ npm run watch

For production, you can run:

$ npm run prod

If you need to compile Modernizr for development only, you can just run npm run modernizr:dev to run the development version.
